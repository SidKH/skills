# Preselection

- Link: https://www.deceptive.design/types/preselection

## Definition

Choose defaults on the user’s behalf in order to steer them toward extra purchases, recurring charges, wider consent, or other provider-preferred outcomes.

## Audit Cues

- checked checkboxes or enabled toggles by default for optional extras
- prefilled cart items, upgrades, or donation frequency selections
- consent or marketing options defaulted to `true`
- defaults that materially benefit the provider and require active user correction
- copy or layout that reduces the chance the user notices the preset choice

## Generic Example

A donation form preselects a recurring contribution option, so users must actively opt out to make a one-time payment.

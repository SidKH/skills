# Confirmshaming

- Link: https://www.deceptive.design/types/confirmshaming

## Definition

Use guilt, shame, or self-belittling language to pressure the user into accepting an offer, subscribing, enabling notifications, or avoiding an opt-out.

## Audit Cues

- dismiss or opt-out buttons with insulting, mocking, or emotionally loaded text
- copy framing refusal as reckless, foolish, uncaring, or self-harming
- A/B copy variants for opt-outs that are emotionally punitive rather than neutral
- dialogs where the accept action is straightforward but the reject action is phrased to make the user feel bad

## Generic Example

A notification prompt offers a positive button and an opt-out labeled with a phrase implying the user does not care about their own wellbeing or is making an obviously bad personal choice.

# Trick Wording

- Link: https://www.deceptive.design/types/trick-wording

## Definition

Use confusing, ambiguous, or misleading language so the user interprets a control or statement one way while the actual effect is different and less favorable.

## Audit Cues

- labels that appear to request one choice while actually authorizing another
- double negatives, ambiguous phrasing, or unusual option ordering
- dropdowns or toggles whose visible framing conflicts with their real consequence
- copy that relies on scan-reading to hide a different meaning
- controls where the opt-out wording is embedded in an unexpected place or label set

## Generic Example

A booking form asks the user to select a residence value from a dropdown, but the only way to decline an add-on is hidden as a non-geographic option inside that same list.

# Visual Interference

- Link: https://www.deceptive.design/types/visual-interference

## Definition

Hide, obscure, or visually downplay important information through typography, contrast, layout, clutter, or unusual placement so users are less likely to notice it.

## Audit Cues

- disclosures in very small, faint, or visually de-emphasized text
- critical terms placed in unexpected regions of the interface
- overloaded layouts where important information competes poorly with promotional elements
- styling that disguises disclosures as secondary chrome rather than core content
- contrast, hierarchy, or spacing choices that make relevant details easy to miss

## Generic Example

An important disclosure appears in low-contrast fine print beneath a large confirmation button, while the rest of the modal emphasizes the provider-preferred action.

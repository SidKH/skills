# Disguised Ads

- Link: [https://www.deceptive.design/types/disguised-ads](https://www.deceptive.design/types/disguised-ads)

## Definition

Present advertisements so they resemble native interface elements, downloads, recommendations, or editorial content, increasing the chance that users click by mistake.

## Audit Cues

- sponsored blocks styled nearly identically to primary content cards or controls
- ad CTAs that reuse native button labels such as `Download`, `Open`, or `Continue`
- ad placements inserted inside lists with minimal disclosure
- tiny or low-contrast sponsorship labels compared with prominent action styling
- components or templates that intentionally reuse native UI for paid placements

## Generic Example

A software page contains a large button that visually matches the genuine download control, but the surrounding markup shows that it is actually a sponsored placement.
